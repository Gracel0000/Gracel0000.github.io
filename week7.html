2026 年 4 月 第 7 週

本週工作摘要

本週的核心任務為架設 CPE Exercise@PU 線上測驗網站，提供靜宜大學學生一個可實際進行 CPE 模擬練習的線上平台，作為後續智慧備考功能整合的基礎環境。

前往 CPE Exercise@PU 網站 ↗

系統架設環境

項目 內容
實體主機作業系統 Windows Server 2022
虛擬化技術 Hyper-V（內建於 Windows Server 2022）
虛擬機作業系統 Ubuntu（Linux）
Online Judge 平台 QDUOJ（青島大學 Online Judge）
容器化部署 Docker / Docker Compose
網站網址 https://cpex.cs.pu.edu.tw/

架設流程說明

1. 主機環境準備

於 Windows Server 2022 實體主機上啟用 Hyper-V 角色，建立一台 Ubuntu 虛擬機作為 QDUOJ 的執行環境，並設定適當的網路橋接，使虛擬機可對外提供服務。

2. QDUOJ 部署

在 Ubuntu 虛擬機上安裝 Docker 與 Docker Compose，使用 QDUOJ 官方提供的 Docker 映像，透過 docker-compose up 快速啟動包含前端、後端、評測沙盒與資料庫在內的完整系統。

3. 網域與反向代理設定

申請並設定 cpex.cs.pu.edu.tw 子網域，透過反向代理將外部 HTTPS 請求導至 Hyper-V VM 內部的 QDUOJ 服務，並完成 SSL/TLS 憑證設定，確保連線安全。

4. 初始化與題庫匯入

完成管理員帳號設定與平台基本組態後，將先前爬取的 49 題 CPE 歷屆題目匯入至本站題庫，確認評測功能正常運作。

本週成果

  • CPE Exercise@PU 網站（https://cpex.cs.pu.edu.tw/）成功上線，可正常存取。
  • QDUOJ 評測引擎於 Hyper-V VM 上穩定運行，可接受程式碼提交並回傳評測結果。
  • 現有題庫已包含 49 題 CPE 歷屆題目，學生可立即進行模擬練習。
  • 網域與 HTTPS 設定完成，確保安全連線。
